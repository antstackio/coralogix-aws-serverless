Please install chromedriver.exe which can be downloaded from here: https://chromedriver.storage.googleapis.com/index.html?path=101.0.4951.41/
and unzip the contents of the zipped file, copy and paste chromedriver into usr/local/bin folder. After that , open terminal
and type in the following commands  'xattr -d com.apple.quarantine /usr/local/bin/chromedriver'
Now run the script.
